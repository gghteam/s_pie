## "어쩌다가 1년 프로젝트가 되 버렸는가" 에 대한 토론
1. 팀장인 우앱이가 심심해서 기획서를 휘갈겨 썼다. <br>
2. 2020 GIGDC에 원레 나가보려고 했지만 그때는 이미 몇몇이 팀이 있는 상태였다. <br>
3. 원레 여름방학에 일해서 출시를 해 보려고 했다. <br><br>
4. 결국 2021 GIGDC 프로젝트가 되었다.<br>
